# AndroidRedPacketDemo
### Description

#### AndroidRedPacketDemo主要演示如何调用相关方法实现红包功能。主要功能如下：

* 红包SDK的集成方式（redpacketlibrary或ARR）
* 获取签名参数
* 进入红包页面（单聊、群聊、小额随机红包）
* 进入转账页面
* 进入零钱页面
* 打开红包
